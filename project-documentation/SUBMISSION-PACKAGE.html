📄 How to Convert HTML to PDF

Easiest Method (Using Browser):

  1. Open any .html file in Google Chrome (recommended) or Firefox/Edge
  2. Press Ctrl + P (Windows) or Cmd + P (Mac)
  3. In the print dialog:
    • Destination: Select "Save as PDF"
    • Enable: "Background graphics" ✓
    • Layout: Portrait
    • Margins: Default
  4. Click "Save" and choose location
  5. Repeat for all chapters

Result: Professional PDF ready for submission! 🎉

Alternative Methods:

  • Microsoft Word: Open HTML in Word → Save As → PDF
  • Online Tool: Visit iLovePDF and upload files
  • Detailed Instructions: Open CONVERSION-GUIDE.html
